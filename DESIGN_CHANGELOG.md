# 🎨 Дизайн: До и После

## Краткое сравнение изменений

### 🔄 Статус подключения

**До:**
- Статичная иконка
- Простой круг с цветом
- Базовая тень

**После:**
- ✨ Пульсирующая анимация для активного соединения
- 🎨 Градиентная иконка
- 💫 SF Symbols bounce эффект
- 🌊 Радиальный градиент с анимацией

---

### 📱 Взаимодействие

**До:**
- Нет тактильной обратной связи
- Простые текстовые сообщения
- Только кнопки для действий

**После:**
- 📳 Haptic feedback для всех действий
- 🎯 Toast-уведомления с иконками
- 👆 Swipe-действия для профилей
- ⬇️ Pull-to-refresh

---

### 🎴 Карточки профилей

**До:**
```
[ ○ ] Профиль
      carrier | transport
      room / client
      [⚡] [🗑️]
```

**После:**
```
┌─────────────────────────────────┐
│ [✓] Профиль                     │ ← Подсветка активного
│     carrier | transport | v1    │ ← Улучшенные pills
│     room / client               │
│                    [⚡] [🗑️]    │
└─────────────────────────────────┘
  ← Swipe влево: удалить
  → Swipe вправо: подключить
```

- Оптимизированные перерисовки

- SF Symbols effects (.bounce, .variableColor)
