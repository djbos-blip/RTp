# ⚡ Быстрый старт с новым дизайном

## 🎯 За 30 секунд

### Что нового?
1. ✨ **Анимации** - пульсирующий статус, плавные переходы
2. 📳 **Вибрация** - тактильная обратная связь на все действия
3. 💬 **Toast** - красивые уведомления сверху экрана
4. 👆 **Swipe** - свайп для быстрых действий
5. ⬇️ **Pull-to-refresh** - потяните вниз для обновления

---

## 🚀 Попробуйте прямо сейчас

### 1. Запустите приложение
```bash
cd OlcRTC-iOS
xcodegen generate
open OlcRTCClient.xcodeproj
# Нажмите Cmd+R для запуска
```

### 2. Проверьте анимации
- Подключитесь к профилю
- Увидите пульсирующий круг вокруг иконки ✨
- Почувствуете вибрацию 📳

### 3. Попробуйте swipe
- Свайпните профиль **влево** → Удалить 🗑️
- Свайпните профиль **вправо** → Подключить ⚡

### 4. Обновите статус
- Потяните список **вниз** ⬇️
- Почувствуете вибрацию
- Статус обновится

### 5. Скопируйте настройки
- Нажмите "socks://"
- Увидите toast-уведомление сверху 💬
- Почувствуете лёгкую вибрацию

## ❓ FAQ

### Почему не работает вибрация?
- Проверьте, что у вас iPhone 6s или новее
- Убедитесь, что haptic feedback включён в настройках iOS

### Почему нет некоторых анимаций?
- Некоторые эффекты требуют iOS 17+
- На iOS 16 будут упрощённые версии

### Как отключить анимации?
- Анимации встроены в систему
- Можно отключить в настройках iOS: Accessibility → Motion → Reduce Motion

### Swipe не работает
- Убедитесь, что свайпаете именно карточку профиля
- Попробуйте свайпнуть медленнее
