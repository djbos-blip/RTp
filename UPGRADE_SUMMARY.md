## 🚀 Как протестировать

### 1. Базовые функции
```bash
# Соберите проект
cd OlcRTC-iOS
xcodegen generate
open OlcRTCClient.xcodeproj
```

### 2. Проверьте анимации
- Запустите приложение
- Подключитесь к профилю → увидите пульсацию
- Смените статус → увидите bounce эффект

### 3. Проверьте haptic feedback
- Нажмите любую кнопку → почувствуете вибрацию
- Скопируйте настройки → лёгкая вибрация
- Остановите соединение → сильная вибрация

### 4. Проверьте swipe-действия
- Свайпните профиль влево → появится кнопка удаления
- Свайпните профиль вправо → появится кнопка подключения

### 5. Проверьте pull-to-refresh
- Потяните список вниз → обновится статус

### 6. Проверьте toast-уведомления
- Импортируйте профиль → увидите зелёный toast
- Скопируйте настройки → увидите toast сверху

## 📱 Требования

### Минимальные
- iOS 16.0+ (базовая функциональность)
- iPhone 6s+ (для haptic feedback)

### Рекомендуемые
- iOS 17.0+ (все анимации и эффекты)
- iPhone 12+ (лучшая производительность)

---

## 🎨 Совместимость

### ✅ Полностью поддерживается
- Dark Mode
- Light Mode
- Dynamic Type
- Accessibility (VoiceOver ready)
- Landscape ориентация
- iPad (адаптивная вёрстка)

### ⚠️ Частично поддерживается
- iOS 16: без некоторых SF Symbols эффектов
- iPhone SE 1st gen: без haptic feedback

---

## 🐛 Известные ограничения

1. **SF Symbols эффекты** (.bounce, .variableColor)
   - Требуют iOS 17+
   - На iOS 16 будут статичные иконки

2. **Haptic Feedback**
   - Работает только на устройствах с Taptic Engine
   - iPhone 6s и новее

3. **Анимации**
   - На старых устройствах могут быть упрощены
   - Для лучшей производительности
