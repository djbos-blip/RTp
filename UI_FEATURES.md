# 📱 Новые UI возможности

## Быстрый гайд по улучшениям интерфейса

### 🎯 Интерактивные жесты

#### Pull-to-Refresh
Потяните список вниз для обновления статуса подключения:
```
     ↓
┌─────────────┐
│   Gateway   │
│      ↓      │  ← Потяните вниз
│   ⟳ ⟳ ⟳    │
└─────────────┘
```

#### Swipe-действия на профилях
```
← Swipe влево
┌─────────────────────┐
│ Профиль       [🗑️]  │  ← Удалить
└─────────────────────┘

→ Swipe вправо
┌─────────────────────┐
│ [⚡] Профиль         │  ← Подключить
└─────────────────────┘
```

### 📳 Haptic Feedback (вибрация)

Приложение теперь вибрирует при действиях:

| Действие | Тип вибрации |
|----------|--------------|
| Копирование | Лёгкая 💫 |
| Вставка | Лёгкая 💫 |
| Удаление профиля | Лёгкая 💫 |
| Обновление | Средняя ⚡ |
| Подключение | Средняя ⚡ |
| Перезапуск | Средняя ⚡ |
| Остановка | Сильная 💥 |
| Ошибка | Сильная 💥 |

---

### 💬 Toast-уведомления

Уведомления появляются сверху экрана:

**Успех (зелёный):**
```
┌─────────────────────────────┐
│ ✓ Профили импортированы     │
└─────────────────────────────┘
```

**Ошибка (красный):**
```
┌─────────────────────────────┐
│ ✗ Ошибка импорта            │
└─────────────────────────────┘
```

**Информация (синий):**
```
┌─────────────────────────────┐
│ ℹ Профиль удалён            │
└─────────────────────────────┘
```

Автоматически исчезают через 2.5 секунды.

### ✨ Анимации

#### Статус подключения
При активном соединении:
- Пульсирующий круг вокруг иконки
- Анимированная точка в бейдже статуса
- Bounce эффект при смене статуса

#### Профили
- Плавное появление новых профилей
- Подсветка активного профиля
- Анимация при подключении/отключении

#### Кнопки
- Плавные переходы disabled/enabled
- Анимация "Скопировано" ↔ "Копировать"
- Spring эффект при нажатии

### 🎨 Цветовая индикация

#### Статусы подключения
- 🟢 **Зелёный** - Запущен
- 🟠 **Оранжевый** - Запускается / Перезапускается
- 🔴 **Красный** - Ошибка
- ⚪ **Серый** - Остановлен

#### Типы логов
- 🔵 **Синий** - Информация
- 🟢 **Зелёный** - Успех
- 🟠 **Оранжевый** - Предупреждение
- 🔴 **Красный** - Ошибка

### 📱 Адаптивность

#### Dark Mode
Все элементы полностью адаптированы:
- Градиенты подстраиваются под тему
- Правильные уровни прозрачности
- Читаемый текст в обеих темах

#### Размеры текста
Поддержка Dynamic Type:
- Автоматическое масштабирование
- `.minimumScaleFactor` для длинных текстов
- Правильные line limits

### ⚡ Производительность

#### Оптимизации
- Lazy loading списков
- Эффективные анимации (60 FPS)
- Минимальные перерисовки
- Правильное использование `@State`

#### Анимации
- Spring animations для естественности
- Response: 0.3-0.4 для быстрых действий
- DampingFraction: 0.8 для плавности

### 🎓 Советы по использованию

1. **Быстрое подключение**
   - Свайп вправо на профиле

2. **Быстрое удаление**
   - Свайп влево на профиле до конца

3. **Обновление статуса**
   - Потяните список вниз

4. **Копирование настроек**
   - Нажмите на нужную кнопку
   - Дождитесь вибрации и toast-уведомления

5. **Импорт профиля**
   - Вставьте ссылку
   - Или нажмите "Вставить" для автоматической вставки

---

### 🐛 Известные особенности

1. **Анимации**
   - Некоторые эффекты требуют iOS 17+
   - На старых устройствах могут быть упрощены

2. **Haptic Feedback**
   - Работает только на устройствах с Taptic Engine
   - iPhone 6s и новее

3. **SF Symbols эффекты**
   - `.bounce` и `.variableColor` требуют iOS 17+
   - На iOS 16 будут статичные иконки
