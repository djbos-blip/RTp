## 🎯 Интерактивные элементы

### Pull-to-Refresh
```
     ↓ Потяните
┌─────────────┐
│   Gateway   │
│      ↓      │
│   ⟳ ⟳ ⟳    │ ← Индикатор загрузки
│             │
│   Content   │
└─────────────┘
```

### Swipe-действия
```
← Swipe влево (удалить)
┌─────────────────────────────┐
│ Профиль              [🗑️]   │
└─────────────────────────────┘

→ Swipe вправо (подключить)
┌─────────────────────────────┐
│ [⚡] Профиль                 │
└─────────────────────────────┘
```

## 🎨 Цветовая легенда

### Статусы
- 🟢 **Зелёный** - Успех, активно, запущено
- 🟠 **Оранжевый** - В процессе, предупреждение
- 🔴 **Красный** - Ошибка, остановка
- 🔵 **Синий** - Информация
- ⚪ **Серый** - Неактивно, остановлено

### Элементы
- 🎨 **Градиент** - Заголовки, иконки
- 💎 **Material** - Toast-уведомления
- 🌈 **Радиальный градиент** - Статус подключения
- ✨ **Тень** - Карточки, панели

## 📱 Анимации

### Пульсация (статус)
```
Кадр 1:  ●
Кадр 2:  ◉
Кадр 3:  ◎
Кадр 4:  ○
Повтор...
```

### Bounce (иконка)
```
Кадр 1:  ✓
Кадр 2:  ✓ (↑ вверх)
Кадр 3:  ✓ (↓ вниз)
Кадр 4:  ✓ (обычно)
```

### Fade in/out (toast)
```
Появление:
0% ░░░░░░░░░░
50% ▒▒▒▒▒▒▒▒▒▒
100% ██████████

Исчезновение:
100% ██████████
50% ▒▒▒▒▒▒▒▒▒▒
0% ░░░░░░░░░░
```

### Scale (профиль)
```
Появление:
0%   ·
50%  ○
100% ●
```
